## What is Guppy?
Guppy is a quantum-first programming language built by the quantum computing company, Quantinuum. The language is embedded into Python which makes reading and writing code simple to learn.
This hybrid approach makes is possible to combine both classical and quantum programming logic.

When writing a Python function, using the `@guppy` decorator can make function run quantum circuits.

```python
# An example of a Guppy Function
@guppy
def circuit() -> qubit:
    q = qubit()
    h(q)
    return q
```


## The Guppy Compiler  
Guppy is strongly-typed, which makes it possible to catch and report compilation errors during development. This helps developers find bugs early and can save time and avoid costly quantum resources.
In addition, the compiler can recognize linearity violoations on qubits. Linearity is an critical part of quantum computing, so just know that this is an essential concept that make quantum algorithms work.

The compiler goes through a series of steps to translate Guppy code into runnable code on quantum hardware, so my goal is to explain how this works for the benefit of future Guppy developers.

In the next parts of the series, I will explain more about why linearity matters and how the Guppy compiler works to enforce this.
